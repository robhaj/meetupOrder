The purpose of this app is to partially automate the process of ordering pizza for Meetups organized through Meetup.com.

By providing a Meetup.com link and specifying the expected percentage of attending RSVPs, the app uses an automated web driver to place an order for the provided meetup, delivered to the event's location.

The web driver utilizes the Chrome browser to automate user input, so you will need to have Chrome installed on your machine.

The order placement function is currently configured to order enough pizzas for approximately two slices per expected attendee. The types of pizzas ordered are approximately distributed as cheese (40%), pepperoni (40%) and veggie (20%).

Pizzas are ordered from Denver Pizza Company, so to successfully place an order, you will need to register a username and password on the DPC website, which you can do here. However, for a demonstration (no order will actually be placed), please use these guest credentials:

  • Username: meetup.pizza@gmail.com
  • Password: galvanize11

Also, payment information is set up locally and will need to be modified to actually place an order.
